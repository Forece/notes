# 快捷键

| 快捷键                   | 作用                           |
| ------------------------ | ------------------------------ |
| CTRL + G                 | 跳转指定行数                   |
| CTRL + \                 | 分屏                           |
| CTRL + ]                 | 相当于 TAB                     |
| SHIFT + TAB 或 CTRL + [  | TAB 反向操作                   |
| Ctrl+C 、 Ctrl+X、Ctrl+V | 复制或剪切当前行/当前选中内容: |
| Shift+Alt+F              | 代码格式化                     |
| Alt+Up 或 Alt+Down       | 向上或向下移动一行             |
| Alt+Shift+Down           | 快速复制当前行到下一行         |
| Ctrl+Shift + K           | 删除当前行                     |
